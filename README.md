# AI Playbook

재사용 가능한 AI 협업 운영 문서, 템플릿, 자동화 설계 문서를 저장하는 개인 playbook 저장소입니다.

## Structure

- `docs/`: 현재 기준으로 유지할 운영 문서
- `notes/`: 실험 메모, 임시 아이디어, 검증 전 초안

## Documents

- `docs/operations/google-drive-migration-plan.md`
- `docs/operations/google-drive-operating-rules.md`
- `docs/checklists/google-drive-migration-checklist.md`

## Rules

- 공통 운영 원칙은 `docs/`에 둡니다.
- 프로젝트 예시나 검증 전 메모는 `notes/`에 둡니다.
- 특정 프로젝트에 강하게 묶이는 내용은 공통 문서 대신 예시 문서로 분리합니다.
